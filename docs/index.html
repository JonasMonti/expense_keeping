Como instalar
  1. Descarrega o APK para o telemóvel.
  2. Abre o ficheiro. O Android vai pedir para permitir instalação de fontes desconhecidas — autoriza.
  3. Confirma a instalação e abre a app. Pronto. 🎉

🍏 iPhone: a Apple não permite instalar por download direto. A app Android é a que se distribui por aqui.
